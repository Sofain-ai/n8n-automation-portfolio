# n8n-automation-portfolio
Professionelles n8n-Automatisierungsportfolio mit API-Integrationen und Datenprozessen.



n8n Automation Portfolio

Workflow: Chat → Email → Google Sheets

Beschreibung:
Dieser Workflow nimmt Chat-Daten entgegen, sendet eine E-Mail und speichert die Daten in Google Sheets.

Features:
- Webhook Trigger
- Email Versand
- Google Sheets Logging
- Fehlerhandling mit Retry
- Saubere Node-Struktur

Setup:
1. Workflow JSON importieren
2. Email Credentials einrichten
3. Google Sheets Credentials verbinden
4. Webhook aktivieren

Tech:
n8n, Webhooks, Google Sheets API
